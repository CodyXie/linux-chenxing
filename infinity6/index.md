# infinity 6

This seems to be an updated version of the infinity3 IP camera SoCs.
The vendor kernel for these seems to be 4.9.84.

## Specs

- 16KB Boot ROM

## SSC323

- Single Cortex A7 with 64MB of DDR2
- Apparently no ethernet.
- [pinout](pinouts.md#88-pin-qfn)
