# IDO-SBC2D86-V1A

Thiny ssd202d based board with a 4 inch screen.

## Hardware

- SigmaStar SSD202D
- FRD720X720BK 4 inch LCD - MIPI?
  - Goodix touchpanel controller
- HYM8563 RTC
- ES7243 - "High Performance Stereo Audio ADC"
- Unisound US516P6
  - LPA4871(?) mono amp
- FS35ND02G - SPI NAND
- SSW101B - Crappy WiFi
- WT5105 - BLE
